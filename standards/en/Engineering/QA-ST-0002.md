This standard is maintained by the QA department.

**[→ View Complete Standard](../QA/QA-ST-0002.md)**

## Quick Summary

QA-ST-0002 defines the minimum requirements for any client request to advance to Architecture.

It covers:
1. Feature / Adjustment / Feature Removal
2. Data Requests
3. Bug Reports
4. Client Questions
5. Deploy Requests
6. Alignment Requests
7. Client Response (Incomplete)
8. Noise
9. New Categories
10. Supreme Rule — "When in Doubt, Ask"

**Executed by:** DJIN Members (QA)
