# QA-ST-0002 — Standard de Qualidade para DJIN Members (QA)

Este standard é mantido pelo departamento de QA.

**[→ Ver Standard Completo](../QA/QA-ST-0002.md)**

---

## Resumo Rápido

QA-ST-0002 define os requisitos mínimos para que qualquer solicitação do cliente possa avançar para Arquitetura.

Ele cobre:
1. Feature / Ajuste / Remoção de Feature
2. Pedidos de Dados
3. Bug Reports
4. Perguntas do Cliente
5. Pedidos de Deploy
6. Pedidos de Alinhamento
7. Resposta do Cliente (Incompleta)
8. Ruído
9. Novas Categorias
10. Regra Suprema — "Na Dúvida, Pergunte"

**Executado por:** DJIN Members (QA)
